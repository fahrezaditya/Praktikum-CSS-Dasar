Font Style

CSS dapat mengatur style dari teks yang pada halaman web yang dibuat. Terdapat beberapa property yang digunakan untuk mengatur style teks yaitu sebagai berikut.

  • text-align: digunakan untuk mengatur perataan teks, value yang dapat digunakan yaitu left, center, dan right
  • color: digunakan untuk memberi warna pada teks
  • letter-spacing: mengatur jarak antar karakter
  • word-spacing: mengatur jarak antar kata
